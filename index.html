你的 AI 怎麼使用工具？

6 種方式，各有取捨。看完這頁你就懂了。

⚡

API

Application Programming Interface
讓程式直接呼叫另一個程式。最快，但要有人寫程式碼來用。
極快 給程式用
🖥️

GUI

Graphical User Interface
包在 API 外面的好看畫面。人類點點按按就能用，但程式不好操控。
慢 給人用
🤖

Browser Use

AI 自己操作瀏覽器
沒有 API？讓 AI 直接操作 GUI。萬能，但很慢很貴。以前叫爬蟲。
很慢 給 AI 用
⌨️

CLI

Command Line Interface
用指令操作。比 GUI 快，比寫程式簡單。指令都是國小英文，LLM 最不容易出錯。
快 給人用 AI 也行
🔌

MCP

Model Context Protocol
統一的工具接口標準。AI 從工具清單選一個就能用。但要等人做、吃 context、生態碎片化。
快 給 AI 用 要等生態
🧠

Skills

AI 學會的技能
教 AI 一次，它自己記住怎麼做。不用等官方、不吃 context、完全客製化。
快 給 AI 用 自給自足
